思考：

  • 检测什么离群点？这个主要和业务相关。数据当中主要包含三方面信息：贷款者基本信息、贷款者信用信息、贷款者还款信息。
    • 在基本信息的属性当中，年收入（annual_inc）以及贷款者月债务总额除以月收入（dti）这两个属性或许是我们离群点分析的重点。
    • 在信用信息的属性当中，我们应当综合多个属性对其进行聚类分析，以找出信用方面的离群点。
    • 在还款信息的属性当中，或许我们应该高度关注还款状态的离群点。
  • 离群点检测方法有很多种，那么你怎么选择呢？
    • 对于连续型属性，在没有类别划分的情况下可以使用z得分来判断离群点。
    • 对于可以划分类别的属性，可以使用聚类或分类算法来找出离群点。

def min_max_uniform(target):
    loan_data[target+'_uniform'] = 0
    mean_target = loan_data[target].mean()
    max_target = loan_data[target].max()
    min_target = loan_data[target].min()
    loan_data[target+'_uniform'] = (loan_data[target] - min_target)/(max_target - min_target)

def outlier_use_zscore(target, threshold):
    loan_data[target+'_is_outlier'] = False
    zscore = (loan_data[target] - loan_data[target].mean())/loan_data[target].std()
    loan_data[target+'_is_outlier'] = (zscore.abs() > threshold)

贷款信息属性

思考良久，我发现，贷款的贷款状态并不是离群点所需要去解决的问题，而是在后续统计当中应该发现的问题。

3. 统计分析¶

那么在此你想分析什么呢？让我们来想想：

  • 基本信息统计：
    • 不同收入等级的贷款者贷款用途有何不同？
    • 不同收入等级的贷款者贷款量如何？
    • 贷款者年收入、工作年限、住房状态有何特征？
    • 美国哪个州贷款人数最多？哪个贷款额度最高？增长率如何？
  • 贷款信息统计：
    • 有哪些类型的贷款更容易吸引投资者投资？
    • 什么样的贷款者喜欢36个月分期付款？60个月的呢？
    • 贷款等级不同的贷款者，他们的信用信息有何不同？
    • 贷款状态不同的贷款，有哪些可以分析的呢？
      • 违约的贷款者有什么特征？
      • 各种状态下的贷款额度有何特点？（来自网络https://www.kaggle.com/erykwalczak/d/wendykan/lending-club-loan-data/initial-loan-book-analysis/discussion）
    • 贷款用途最多的是哪一种？最少的呢？
    • 投资回报率最高的贷款具有什么特征？
